发表于2020-05-08|更新于2022-08-05|gitutterances
|字数总计:382|阅读时长:1分钟|阅读量:

utterances 是一款基于 GitHub issues 的评论工具。

相比同类的工具 gitment、gitalk 以及 disqus 评论工具，优点如下：

1.极其轻量
2.加载非常快
3.配置比较简单

1.utterances 安装

github 安装地址：https://github.com/apps/utterances

首先安装这个 App ，选择要关联评论的仓库。可以选择所有仓库，也可以只选择一个仓库。选择一个仓库比较安全

3.utterances配置

在themes/yilia/_config下添加以下代码：

enable：true使用utterances评论false关闭
repo： 你的用户ID/仓库名称，如：blog/utterance_repo
issue_term：映射配置,有如下选项：

  1. pathname
  2. url
  3. title
  4. og:title
  5. issue-number
  6. specific-term

theme：主题,有如下选项：

  1. github-light
  2. github-dark
  3. github-dark-orange
  4. icy-dark
  5. dark-blue
  6. photon-dark
